Bangladeshi payment gateways for CampTix

== Description ==

CampTix Bangladeshi Payment Gateway Solution for Camptix Plugin. Allow visitors to purchase tickets to your online or offline event using Bangladeshi payment gateway, directly from your WordPress or WordCamp website.

= Supported Gateways =

 * [aamarPay](https://www.aamarpay.com/) - `BDT` currency

= About aamarPay =

[aamarPay](https://www.aamarpay.com/) is an Online Payment Gateway & Merchant Service Provider of Bangladesh. Please contact them to obtain the credentials to make the gateway work. This payment gateway is based on the [documentation](https://www.aamarpay.com/developer/) of their official site.

= Contribute =

Feel free to post your feature requests, issues and pull requests to [Camptix Bangladeshi Payments](https://github.com/tareq1988/camptix-bd-payments) on GitHub.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/bd-camptix-payments` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the Camptix -> Setup screen to configure the plugin

== Frequently Asked Questions ==

= Which payment gateway it supports =

Right now it supports [aamarPay](https://www.aamarpay.com/) gateway.

= Which currencies does it support? =

Right now it supports `BDT` currency only.
